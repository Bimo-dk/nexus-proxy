# nexus-proxy

## 1.0.0

First stable release. `nexus-proxy` is the dev-time hot-reload proxy.
Most developer workflows now go through `bnx dev` (which embeds the
same proxy in-process); this standalone package remains for setups
that need a long-running proxy outside the CLI.
